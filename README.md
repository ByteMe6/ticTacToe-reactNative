# Tic Tac Toe

A Tic Tac Toe game built with React Native. Runs on Android, iOS, and in the browser.

[Русский](./README.ru.md)

## Features

- Classic 3×3 Tic Tac Toe
- Game state saved between sessions (AsyncStorage)
- Runs on Android, iOS, and Web (react-native-web)

## Run

### Web

```bash
npm install
npm run web
```

### Android

```bash
npm run android
```

### iOS

```bash
npm run ios
```

> Requires React Native environment setup: https://reactnative.dev/docs/environment-setup

## Build web & deploy

```bash
npm run deploy
```

Deploys to GitHub Pages.

## Tech

- React Native 0.74
- TypeScript
- react-native-web
- AsyncStorage
- Vite (web build)
